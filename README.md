# Chat App Demo

This is a simple chat app demo using React and Socket.io, styled with shadcn/ui.

### Requirements

Node 18

### Running the app

```bash
npm install
npm run dev
```

The app will be running on http://localhost:5173/

You can also run the app with Docker:

```bash
docker compose up -d
```
